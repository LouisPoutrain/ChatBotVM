# PLAN DE GESTION DE DONNEES (PGD / DMP)

# Les données de recherche

Les données sont obtenues tout au long du travail de recherche.

Pour pouvoir les utiliser correctement, il est important de les décrire et de les comprendre en se posant certaines questions :

- de quel type de données s'agit-il?
- comment ont-elles été produites ?
- dans quel contexte peuvent-elles être réutilisées?

# Certaines données doivent être conservées

avec des règles d'accès précises pour des raisons éthiques, juridiques ou commerciales

# D'autres, en revanche, peuvent être partagées plus largement.

Leur mise à disposition permet de favoriser les échanges, la collaboration et la progression des connaissances au sein de la communauté scientifique.

Vous l'aurez compris, pour savoir comment gérer ses données il vous faut un plan :

# le Plan de Gestion de Données (PGD).

- Il vous aide à identifier et caractériser les données que vous allez produire,
- Définir comment les organiser, les stocker, les traiter
- et déterminer celles que vous allez pouvoir partager.

### Pourquoi un PGD?

Le PGD s'inscrit dans une démarche de science ouverte et est un bon outil pour la gestion de projet.
