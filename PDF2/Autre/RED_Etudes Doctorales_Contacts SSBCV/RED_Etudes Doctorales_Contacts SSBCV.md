# Note d'Information pour les Doctorants SSBCV

Dans le cadre de votre parcours doctoral, il est essentiel de savoir que vous avez plusieurs interlocuteurs à votre disposition pour discuter de vos préoccupations, qu'elles soient d'ordre administratif, personnel, ou liées à l'avancement de votre thèse. Cela peut inclure des questions sur l'encadrement, les ressources matérielles de laboratoire ou tout autre aspect impactant votre progression. Voici les contacts importants pour chaque site de l'ED SSBCV : Université de Tours, Université d'Orléans.

Pour toute question ou assistance, n'hésitez pas à contacter les interlocuteurs listés ci-dessous.
