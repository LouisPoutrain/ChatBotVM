## Le Règlement Général sur la Protection des Données (RGPD / GPDR)

Le RGPD permet d'encadrer les données collectées dans le cadre de la recherche:

- Comment les collecter
- Comment les stocker
- Comment les partager

## Qu'est-ce qu'une donnée personnelle?

on peut définir une donnée personnelle comme toute information se rapportant à une personne identifiée ou identifiable (Article 4 du RGPD)

Exemples de données concernées :

- Annotations et post-it
- Captures d'écran
- Enregistrements vocaux
- Transcriptions de réunions
