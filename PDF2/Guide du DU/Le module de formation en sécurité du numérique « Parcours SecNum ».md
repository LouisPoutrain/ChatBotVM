# Le module de formation en sécurité du numérique « Parcours SecNum »

**Il existe deux types d’organisations : celles qui ont été piratées et celles qui ne savent pas encore qu’elles ont été piratées**.

Cela ne signifie pas que tout est perdu, bien au contraire. Cela doit nous inciter à reprendre le contrôle. L’équipe cybersécurité de l’université vous propose plusieurs fois par an une journée de formation, disponible dans l’ENT (GeFORP) pour vous permettre d’adopter les bons réflexes pour protéger vos données.

#### **Public(s) cible(s)**

Tous publics

#### **Programme**

Module 1 : Sécuriser ses données, par où commencer ?

* Comment se protéger du phishing
* Gérer ses stockages amovibles
* Sauvegarder ses données
* Se protéger des virus
* Travailler en mobilité, chez soi ou à l’étranger, les bons réflexes
* Les machines d’expérimentation

Module 2 : Gérer l’authentification au quotidien

* Principes de l’authentification
* Attaques sur les mots de passe
* Sécuriser ses mots de passe
* Gérer ses mots de passe avec un coffre-fort de mot de passe (outil fourni)

Module 3 : Protéger les données sensibles

* Déterminer la sensibilité de ses données
* Utiliser les bons outils et support de stockage
* Utiliser un outil de chiffrement des données (outil fourni)
